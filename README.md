# SWOW-DE-2025-Code

This repository includes the code used to preprocess and analyze the SWOW-DE-2025 data in below-mentioned publication.

## Data availability

If you are interested in the SWOW-DE data, please visit the Small World of Words project page: https://smallworldofwords.org/en/project/research

## Information on this repository

Please note that not all external data can be shared in this repository due to file size and licencing limitations. 

This repository thus contains readme and source text files that detail which original data from what source should go where instead.
